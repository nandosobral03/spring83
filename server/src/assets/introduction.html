Server Disposition

This server runs on a Linode VPS with 1GB of RAM and 1 CPU core, can't really make any promises about uptime or performance, but it's been running for a year now without any issues.
This server stores boards for 21 days.

Standards for publishing

This server has certain restrictions in place to ensure a positive and safe environment. Firstly, it prohibits boards that contain abusive or discriminatory language or imagery. Secondly, it does not allow boards that promote or link to resources supporting abuse or discrimination. Additionally, engaging in illegal activities is strictly forbidden.

If you come across any boards that violate these rules, you can report them using the provided address. I reserve the right to remove any boards that violate these rules.

The server follows the 200220629 standard for publishing.
